CS 371

In-class 1:
Write a basic web client transaction

Write the client side of a very basic web transaction.

Before You Begin

  1. Make sure you've read and understood this I/O Streams readme.txt and sample code posted on GitHub. If you don't understand this code, stop and ask questions.
  2. Read through Java's tutorial on Reading and Writing to a Socket. Again, if the sample code here doesn't make sense, ask questions before moving on.

Details

Write a program that will

  1. Connect to a web server
  2. Request a single document
  3. Print the response headers on the standard output
  4. Save the payload (the requested document) to a file.

Verify that your program works for

  • http://www.cis.gvsu.edu/~kurmasz/Humor/stupid.html
  • http://www.cis.gvsu.edu/~kurmasz/Distiller/HowTo.txt
  • http://www.cis.gvsu.edu/~kurmasz/NoSuchFile.html
  1. How can you tell from the response headers that the request for NoSuchFile.html failed?
  2. What (if anything) was written to the file? Where did this data come from?

Now try fetching http://www.cis.gvsu.edu/~kurmasz/buzz1.jpg.

  1. Did it work? Why not?
